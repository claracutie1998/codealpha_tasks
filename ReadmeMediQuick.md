# MediQuick – UI/UX Design Case Study (Figma)

MediQuick is a mobile healthcare and medicine delivery application designed in Figma to provide users with a fast, reliable, and user-friendly way to order medications, upload prescriptions, track deliveries, and manage healthcare needs from their smartphones.

## Project Overview

This project was created as a complete high-fidelity UI/UX design exercise, transforming low-fidelity wireframes into a polished and interactive mobile application. The goal was to design a modern healthcare experience that prioritizes accessibility, simplicity, and trust while maintaining a clean and professional visual identity.

## Design Process

### 1. User Research & Planning

* Identified key user needs for medicine ordering and prescription management.
* Defined the primary user flow from onboarding to order delivery.
* Created user journey maps to understand the customer experience.

### 2. Wireframing

* Developed low-fidelity wireframes to establish screen layouts and navigation structure.
* Focused on information hierarchy and user flow before applying visual styling.
* Designed core screens including onboarding, authentication, home dashboard, product listing, cart, checkout, order tracking, and profile management.

### 3. High-Fidelity UI Design

The wireframes were transformed into a modern healthcare interface by incorporating:

* Consistent color palette focused on trust and reliability.
* Modern typography with clear visual hierarchy.
* Custom icons and illustrations.
* Reusable components and design system elements.
* Rounded cards, buttons, and input fields for a modern mobile experience.
* Accessibility-focused spacing and touch-friendly interactions.

### 4. Design System

Created a scalable design system including:

* Primary Color: #3B82F6 (Medical Blue)
* Secondary Color: #60A5FA
* Neutral Colors: White, Light Gray, Dark Gray
* Consistent spacing system
* Button variants
* Input components
* Card components
* Navigation elements
* Icon library

### 5. Prototyping & Interaction Design

Built interactive prototypes in Figma to simulate:

* Onboarding flow
* User authentication
* Product search and browsing
* Cart management
* Checkout process
* Order tracking experience
* Profile navigation

Interactive transitions and micro-interactions were added to create a realistic user experience.
