## Design Process

Zuri Travel was designed in **Figma** with a strong focus on creating a modern, intuitive, and visually engaging travel booking experience. The design process began with user flow planning, wireframing, and information architecture to ensure a seamless journey from onboarding to hotel booking.

The interface features a **dark-themed UI** with vibrant blue accents, premium travel imagery, and clean typography to create a luxury travel experience. High-fidelity screens were designed for onboarding, authentication, destination discovery, hotel details, and profile management.

Interactive prototypes were created using **Figma Smart Animate**, overlays, and micro-interactions to simulate real-world mobile app behavior. Special attention was given to usability, accessibility, visual hierarchy, and smooth navigation patterns inspired by leading travel platforms such as Airbnb and Booking.com.

### Key Design Deliverables

* User flow mapping
* Low-fidelity wireframes
* High-fidelity mobile UI designs
* Interactive Figma prototype
* Component-based design system
* Reusable buttons, cards, inputs, and navigation elements
* Dark mode design with consistent spacing and typography
* Hotel discovery and booking experience

### Tools Used

* Figma
* Auto Layout
* Components & Variants
* Smart Animate
* Interactive Prototyping

The final design delivers a polished mobile experience that enables users to discover destinations, explore accommodations, and book travel experiences through a simple and elegant interface.
